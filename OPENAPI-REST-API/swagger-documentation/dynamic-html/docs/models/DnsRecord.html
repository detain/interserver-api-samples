DnsRecord

  • id : String
    The ID of the DNS Record.
  • domain_id : String
    The ID of the Domain this is a record of.
  • name : String
  • type : DnsRecordType
  • content : String
    The content of the record, such as the IP address or hsotname.
  • ttl : String
    Time To Live (seconds)
  • prio : String
    Priority
  • disabled : String
  • ordername : String
    Alternate name to use for sorting
  • auth : String
