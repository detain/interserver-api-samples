This is the API

addVps

Places an order for a new VPS. Use `PUT /vps/order` to validate the order first.

URL

https://my.interserver.net/apiv2/vps/order

HTTP Method

POST

Response Type

ServiceOrderPostResponse

Parameters

  • Body: VpsOrderPostRequest VpsOrderPostRequest(VpsOrderPostRequest)

deleteVpsBackup

Permanently removes the specified backup file from storage. Use `GET /vps/{id}/backups` to list available backup filenames before deleting.

URL

https://my.interserver.net/apiv2/vps/{id}/backups

HTTP Method

DELETE

Response Type

SuccessTextResponse

Parameters

  • Path: id Integer

    VPS ID number

  • Query: file String

    The backup filename to delete.

  • Query: all String

    Set to `1` to list all backups across all services, not just the ones for the given VPS.
