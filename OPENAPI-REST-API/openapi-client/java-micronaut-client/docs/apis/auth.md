# Authorization methods
<a id="name"></a>
# apiKeyAuth
Authorization method is of **api key type**. The parameter `X-API-KEY` is located in `header`.

It can be configured in the [application.yml](src/main/resources/application.yml) using Micronaut security features.
<a id="name"></a>
# sessionIdHeaderAuth
Authorization method is of **api key type**. The parameter `sessionid` is located in `header`.

It can be configured in the [application.yml](src/main/resources/application.yml) using Micronaut security features.

# sessionIdCookieAuth
Authorization method is of **api key type**. The parameter `sessionid` is located in `cookie`.

It can be configured in the [application.yml](src/main/resources/application.yml) using Micronaut security features.
